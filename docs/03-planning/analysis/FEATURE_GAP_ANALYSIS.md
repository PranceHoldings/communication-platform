### 📊 ベンチマークシステム（BENCHMARK_SYSTEM）

**現状:** Phase 4.2に概要のみ記載（1.5週間）

**詳細化が必要な機能:**
- プロファイル算出アルゴリズム
  - K-means クラスタリング
  - プロファイル定義（年齢層、職種、業界、経験年数等）
- パーセンタイル計算
  - 同シナリオの全ユーザーとの比較
  - 業界別・ポジション別ベンチマーク
- 成長トラッキング
  - 時系列スコア変化グラフ
  - 改善率計算
  - マイルストーン達成
- パーソナライズド改善提案
  - AI駆動の改善提案（Claude API）
  - 弱点カテゴリの特定
  - 推奨トレーニングシナリオ
- プライバシー保護
  - 完全匿名化（個人特定不可）
  - 最小人数制限（N≥10）

**データモデル追加:**
```prisma
model BenchmarkProfile {
  id              String   @id @default(cuid())
  userId          String   @unique
  scenarioType    String
  totalSessions   Int
  averageScore    Float
  categoryScores  Json
  percentile      Int
  cluster         String?
  lastUpdatedAt   DateTime
}
```

**API実装:**
- `GET /api/v1/benchmark/profile`
- `GET /api/v1/benchmark/percentile`
- `GET /api/v1/benchmark/growth`
- `GET /api/v1/benchmark/suggestions`

**推奨実装期間:** Phase 4.2を2週間に延長（Week 17-18）
