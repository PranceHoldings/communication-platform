# リリースロードマップ - Prance Communication Platform

**作成日:** 2026-03-09
**バージョン:** 1.0
**ステータス:** 最新版

---

## 📋 概要

本ドキュメントは、Pranceプラットフォームの**リリース戦略**を定義します。4段階のリリースフェーズで、段階的に機能を提供し、市場フィードバックを反映します。

---

## 🎯 リリース戦略

### 基本方針

1. **MVP First:** 最小限の機能で早期リリース、市場検証
2. **顧客フィードバック重視:** Beta Release後、顧客ニーズに応じて調整
3. **段階的価値提供:** 各リリースで明確な価値を提供
4. **品質優先:** 機能追加よりも品質・安定性を優先

### リリースフェーズ

```
MVP Release (Internal)
    ↓ (2ヶ月後)
Beta Release (Limited External)
    ↓ (2ヶ月後)
V1.0 General Availability (Public)
    ↓ (2ヶ月後)
V2.0 Enterprise Edition (Enterprise)
```

## 🚀 MVP Release - 基本会話・録画・解析

**リリース日:** 2026年4月末（目標）

**ステータス:** 🔄 進行中（Phase 2.2, 2.3実装中）

### 目標

AIアバターとの会話セッション、録画、基本解析が動作する最小限の製品をリリースし、**内部テスト**と**初期顧客テスト**を実施。

### 対象ユーザー

- **内部ユーザー:** CLIENT_ADMIN, CLIENT_USER（自社社員、パートナー企業）
- **テストユーザー:** 5-10社の初期顧客（NDA締結）

### 含まれる機能

#### コア機能
- ✅ ユーザー認証・管理
- ✅ シナリオ管理（CRUD + Clone）
- ✅ アバター管理（CRUD + Clone）
- ✅ 音声会話セッション（STT → AI → TTS）
- ✅ リアルタイムWebSocket通信
- ✅ 多言語対応（10言語）

#### 録画・解析機能
- ✅ セッション録画（アバター + ユーザーカメラ）
- 🔄 表情・感情解析（AWS Rekognition）
- 🔄 音声特徴解析（音高・速度・フィラーワード）
- 🔄 スコアリング（総合スコア + カテゴリ別スコア）
- 🔜 レポート生成（PDF）
- 🔜 AI改善提案

### 含まれない機能（将来のリリースで提供）

- ❌ ゲストユーザーアクセス（Beta Releaseで提供）
- ❌ XLSX一括登録（Beta Releaseで提供）
- ❌ ATS連携（Beta Releaseで提供）
- ❌ サブスクリプション課金（V1.0 GAで提供）
- ❌ 外部API（V1.0 GAで提供）

### 成功基準

**技術指標:**
- [ ] セッション完了率 > 95%
- [ ] 録画成功率 > 98%
- [ ] 解析処理時間 < 60秒（1分セッション）
- [ ] レポート生成時間 < 30秒
- [ ] システム稼働率 > 99%

**ビジネス指標:**
- [ ] 5社以上の初期顧客テスト実施
- [ ] NPS（Net Promoter Score） > 30
- [ ] 顧客満足度 > 4.0/5.0
- [ ] 重大バグ 0件

### デリバラブル

1. **製品:**
   - Pranceプラットフォーム（MVP版）
   - 管理者ダッシュボード
   - セッションプレイヤー
   - レポートビューア

2. **ドキュメント:**
   - ユーザーマニュアル（日本語・英語）
   - 管理者ガイド
   - API仕様（内部用）

3. **マーケティング:**
   - 製品紹介スライド
   - デモビデオ

- **内部ユーザー:** CLIENT_ADMIN, CLIENT_USER
- **ゲストユーザー:** 外部候補者、研修受講者（ログイン不要）
- **Beta顧客:** 10-20社（採用担当者、採用代行企業）

### MVP Releaseからの追加機能

#### ゲストユーザーシステム（Phase 2.5）
- 🆕 ゲストセッション作成UI

- 🆕 ログイン不要アクセス
- 🆕 自動録画・評価
- 🆕 データアクセス制限

#### XLSX一括登録（Phase 3.1.1）
- 🆕 XLSXテンプレート提供
- 🆕 一括アップロード・バリデーション
- 🆕 バッチ処理（1000件/5分）
- 🆕 自動メール送信（SES統合）

#### 基本ATS連携（Phase 3.1.2）
- 🆕 Greenhouse統合
- 🆕 Lever統合
- 🆕 Workday統合
- 🆕 候補者同期（ATS → Prance）
- 🆕 結果エクスポート（Prance → ATS）

#### 基本レポート・分析（Phase 3.1.3）
- 🆕 候補者検索・フィルタ
- 🆕 Excelエクスポート
- 🆕 基本分析ダッシュボード
- 🆕 スコア分布グラフ

### ビジネス価値

**運用効率化:**
- **90%の時間削減:** 手動入力 → XLSX一括登録
- **80%のエラー削減:** ATS連携による二重入力排除
- **候補者体験向上:** ログイン不要アクセスで離脱率低減

**スケーラビリティ:**
- 1000人の候補者を5分で登録
- ATSからの自動同期で人的リソース不要

### 成功基準

**技術指標:**
- [ ] XLSX処理成功率 > 95%
- [ ] ATS同期成功率 > 98%
- [ ] ゲストアクセス成功率 > 99%

**ビジネス指標:**
- [ ] 10社以上のBeta顧客獲得
- [ ] 月間500+ゲストセッション実施
- [ ] 顧客継続率 > 80%
- [ ] NPS > 40

### デリバラブル

1. **製品:**
   - ゲストユーザーアクセス画面
   - XLSX一括登録機能
   - ATS連携管理画面
   - 検索・分析ダッシュボード

2. **ドキュメント:**
   - XLSX一括登録ガイド
