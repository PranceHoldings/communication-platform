## プロジェクト概要

### コンセプト

**AIアバターとのリアルタイム会話を通じて、面接練習・語学学習・企業研修を実現するマルチテナント型SaaSプラットフォーム**

- 🎯 **目的**: 人材育成・評価プロセスのデジタル化と標準化
- 🌐 **展開形態**: クラウドベースSaaS（初期投資不要）
- 📈 **スケール**: 10名〜10万名まで対応可能
- 🌍 **多言語**: 日本語・英語（初期）、将来的に8言語以上

### なぜPranceなのか？

| 従来の方法 | Pranceの解決策 |
|-----------|---------------|
| 面接官・トレーナーの確保が困難 | AIアバターが24時間365日対応 |
| 評価基準が属人的・バラバラ | 標準化された評価基準とレポート |
| フィードバックまで時間がかかる | セッション終了後すぐに詳細レポート |
| 大人数の研修はコスト高 | 同時100名以上のセッション実行可能 |
| 遠隔地からの参加困難 | 完全オンライン、場所を選ばない |

## 主要機能

### 🤖 1. AIアバター会話（コア機能）

**リアルタイムUI: 3要素同時表示**

```
┌─────────────────────────────────────────────┐
│  AIアバター映像  │  あなた（カメラ映像）   │
│  リアルタイム    │  リアルタイム          │
│  レンダリング    │  カメラ取得            │
├─────────────────────────────────────────────┤
│  リアルタイム文字起こし（会話履歴）        │
│  00:12 AI: よろしくお願いします。          │
│  00:18 YOU: よろしくお願いします。私は...  │
│  00:34 AI: 技術スタックについて...         │
│  話者別色分け、自動スクロール              │
└─────────────────────────────────────────────┘
```

**特徴:**
- ✅ **自然な会話**: Claude AI（最先端の対話AI）による人間らしい応答
- ✅ **低レイテンシ**: 50-200msの応答速度、違和感のない会話
- ✅ **完全同期**: 音声とリップシンクの完全一致
- ✅ **リアルタイム字幕**: Azure STTによる即座の文字起こし

### 📹 2. 録画・解析

**自動録画:**
- ユーザーとAIアバターの映像を同時録画
- サイドバイサイド合成動画を自動生成
- 録画保存期間: プラン別（7日〜無制限）

**自動解析:**
- 😊 **感情解析**: 7種類の感情（喜び・悲しみ・怒り等）を時系列で分析
- 👁️ **非言語行動**: アイコンタクト、姿勢、視線方向を評価
- 🗣️ **音声特徴**: 話速、声のトーン、無音区間、フィラーワード検出
- 📝 **会話内容**: AIによる内容評価、キーポイント抽出

### 📊 3. ベンチマーク・成長トラッキング

**あなたの位置づけを可視化:**

```
総合スコア: 78/100
全体平均: 65/100
→ あなたは上位 23% です！

項目別比較:
論理的説明力    ████████░░  78  (平均: 65)
アイコンタクト  ██████████  90  (平均: 72) ⭐優秀
話速・間合い    █████░░░░░  58  (平均: 68) ⚠要改善
```

**パーソナライズド改善提案:**
- AIが個人の弱点を分析し、具体的な改善アクションを提案
- 成長推移グラフで進捗を可視化（6ヶ月間）
- 達成バッジでモチベーション維持

### 🔌 4. 外部連携

**ATS（採用管理システム）連携:**
- 国内主要3社: HRMOS採用、ジョブカン採用管理、採用一括かんりくん
- 海外主要3社: Greenhouse、Lever、Workday Recruiting
- 候補者情報の自動同期、評価結果の自動送信

**API統合:**
- RESTful API提供（外部システムから利用可能）
- Webhook通知（イベント駆動連携）
- プラン別APIコール数制限

### 🎨 5. カスタマイズ

**アバター:**
- 50種類以上のプリセットアバター（2D/3D、アニメ/リアル）
- ユーザー画像からのカスタムアバター生成（Pro以上）
- 組織専用アバター登録（Enterprise）

**シナリオ:**
- プリセットシナリオ（面接、語学、接客等）
- 独自シナリオ作成（質問内容、評価基準をカスタマイズ）
- AIプロンプト管理（Enterprise: 会話ロジックを完全制御）

**レポート:**
- 標準レポートテンプレート
- カスタムレポートテンプレート（Enterprise）
- PDF自動生成・ダウンロード

## ユースケース

### 🎓 1. 就職・採用支援

**課題:**
- 面接経験不足で本番で緊張してしまう
- 面接官の確保が難しい、評価が属人的

**Pranceの解決策:**
- 何度でもAIアバターと模擬面接
- 標準化された評価基準で公平な評価
- 詳細なフィードバックで改善点が明確

**導入事例（想定）:**
- 大学キャリアセンター: 学生1,000名が年間5,000セッション実施
- 採用代行企業: 複数クライアント候補者を一元管理・評価

### 🌍 2. 語学学習

**課題:**
- ネイティブスピーカーとの会話機会が少ない
- 発音や表現の適切さがわからない

**Pranceの解決策:**
- AIアバターネイティブスピーカーと24時間会話練習
- 発音・語彙・流暢さの自動評価
- ハイライトシーン（良い表現・改善箇所）の自動抽出

### 🏢 3. 企業研修

**課題:**
- カスタマーサービス・営業のロールプレイが非効率
- 全員に均一な研修を提供できない

**Pranceの解決策:**
- クレーム対応、営業トークのシナリオでAI相手に練習
- 全受講者の評価を一覧で比較・分析
- 録画を見返して改善点を確認

## プラットフォームの特徴

### 💡 技術的優位性

| 特徴 | 説明 | ビジネス価値 |
|------|------|-------------|
| **マルチテナント型SaaS** | 単一インフラで数千組織を管理 | 初期投資不要、即座に利用開始 |
| **サーバーレス** | AWS Lambda、Aurora Serverless v2 | 自動スケール、使用量課金でコスト最適化 |
| **AI最新技術** | Claude Opus 4（最先端の対話AI） | 自然な会話、高精度な評価 |
| **リアルタイム処理** | 50-200ms応答、60fps描画 | 違和感のない会話体験 |
| **高可用性** | 99.9% SLA、マルチAZ配置 | 安定したサービス提供 |

### 🔒 セキュリティ・プライバシー

- 🛡️ **暗号化**: 全データAES-256暗号化（保存時・通信時）
- 🔐 **認証**: OAuth2/SAML SSO対応（Enterprise）
- 👥 **テナント分離**: 完全なデータ分離（Row Level Security）
- 📋 **コンプライアンス**: GDPR、個人情報保護法対応
- 🗑️ **データ削除**: ユーザーによるデータ完全削除可能

### 🌏 グローバル対応

- 🌐 **多言語UI**: 日本語・英語（初期）、将来8言語以上
- 🗣️ **音声認識**: 40言語以上対応（Azure STT）
- 🎤 **音声合成**: 多言語ネイティブ音声（ElevenLabs）
- 🌍 **グローバルCDN**: CloudFrontで世界中から高速アクセス
